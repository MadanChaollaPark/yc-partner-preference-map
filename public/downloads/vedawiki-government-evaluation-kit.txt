Vedawiki Government Evaluation Kit
Version: public manifest, May 2026

Vedawiki builds a passive controller-side and log-import flight recorder for
authorized non-weaponized drone evaluation, training debriefs, and manufacturer
QA programs.

This public kit is a manifest, not an unrestricted operational binary. Signed
tool builds and device images are released only to authorized public-sector,
research, or approved manufacturer contacts after onboarding.

Current prototype
- Records USB HID / Xbox-compatible gamepad input.
- Saves timestamped controller events.
- Imports approved passive CSV/JSON telemetry logs.
- Lists supported source families through the local API.
- Exports structured logs for review sessions.
- Starts at the controller side so mixed fleets can be evaluated before each
  drone model has a deep telemetry integration.

Evaluation package available after onboarding
- Signed logger build.
- Supported controller list.
- Passive USB/import and read-only serial deployment plan.
- Pilot deployment checklist.
- Data handling and chain-of-custody worksheet.
- Debrief viewer / API documentation.

Data posture
- Explicit authorization required.
- Local-first capture supported.
- No informal mission scraping.
- No protected-link decoding, firmware modification, or command injection.
- Exports are intended for approved review, training, QA, or procurement
  evaluation workflows.
